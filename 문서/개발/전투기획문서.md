## 1. 게임 구조 개요

### 기본 흐름

```
[판 시작]
    ↓
[스타팅 캐릭터 선택] — 2~3명 중 1명 선택
    ↓
[경영 단계] — 아침  ※별도 문서
    ↓
[탐사 단계] — 저녁
    ↓
[결산] → Day 증가
    ↓
[반복]
    ↓
[스토리 분기 or 판 종료]
```

- Day 카운트가 게임 진행의 척도
- 탐사 실패(전멸) = 파티 전원 영구 사망
- 판이 끝나도 기록칩은 영구 유지

---

### 자원 종류

| 자원 | 획득처 | 용도 |
|------|--------|------|
| 화폐 | 낚시 이벤트, AI 처치 보상 | 캐릭터 영입, 경영 |
| 고철 | 잔해 수집 이벤트, 잠수부 처치 보상 | AI 제작, 수리 |
| 심해화폐 | 탐사 보상, 특수 이벤트 | 심해 상점 전용 |
| 아이템 | 잠수부 처치 보상, 이벤트, 상점 | 캐릭터 장착 |

## 2. 캐릭터 시스템

### 캐릭터 구조

- 일러스트 100종 고정 풀
- 영입 시 직업·스탯·스킬·이름 전부 랜덤 생성
- 같은 얼굴이지만 매번 다른 사람이 태어나는 구조
- 기록칩은 일러스트(얼굴)에 귀속 → 판을 넘어 누적

### 일러스트 구성 (캐릭터당 5장)

| 종류 | 사용 시점 |
|------|-----------|
| 일반 | 기본 상태 |
| 웃음 | 이벤트, 대사 |
| 눈감음 | 피격, 감정 |
| 사망 | 전투불능, 영구사망 |
| 스킬사용 | 사용 스킬 발동 시 |

### 직업

| 직업 | 전투 특성 | 처치 보상 | 영입 방법 |
|------|-----------|-----------|-----------|
| 낚시꾼 | 전투 특화. 공격력↑ 피해감소 | 없음 | 화폐로 영입 |
| 잠수부 | 피해 증가 | 아군 처치 시 아이템 | 화폐로 영입 |
| AI | 피해 증가 | 아군 처치 시 화폐 | 고철로 제작 |

- 잠수부·AI는 직접 처치 불필요. 파티 내 누군가가 처치하면 보상 발동
- 낚시꾼 위주 파티: 전투력↑ 수익↓
- 직업 혼성 파티: 전투력↓ 수익↑
- 파티 구성이 핵심 딜레마

### 스탯

| 스탯 | 설명 |
|------|------|
| 체력 | HP |
| 건강 | 라운드/탐사 종료 시 자연 회복량 |
| 공격 | 공격력 |
| 민첩 | 공격속도 |
| 행운 | 크리티컬 확률 + 회피율 |

### 코그 등급

- 캐릭터 생성 시 1~10 중 확정, 이후 고정
- 아이템 장착으로 소폭 보정 가능
- 등급이 높을수록 스탯 합산 범위 상승
- 코그 10은 네임드 캐릭터 전용

### 스킬 구조

**일반 스킬** — 자동 발동. 조건 기반.
- 예: 공격 3회마다 발동 / 피격 시 발동 / 체력 50% 이하일 때 발동

**사용 스킬** — 플레이어가 초상화 클릭 시 수동 발동.
- 게이지가 요구 수치에 도달해야 사용 가능
- 발동 시 해당 캐릭터 일러스트가 스킬사용 컷으로 전환

### 게이지 시스템

- 캐릭터별 독립 게이지
- 충전 조건: 타격 + 피격 횟수 합산
- 게이지는 라운드 종료 시 유지, 탐사 종료 시 초기화
- 사용 스킬마다 요구 게이지 수치 다름

| 스킬 등급 | 요구 게이지 |
|-----------|-------------|
| 약 | 100 |
| 중 | 200 |
| 강 | 300 |

- 낚시꾼: 타격 多 → 게이지 빠름 → 사용 스킬 자주 발동
- 잠수부·AI: 타격 少 + 피격 多 → 게이지 느림 → 사용 스킬 가끔

## 3. 기록칩 시스템

### 개요

- 일러스트(얼굴)에 귀속되는 영구 누적 기록
- 판을 넘어 유지. 설정에서 직접 초기화하지 않는 한 삭제 불가
- 같은 얼굴이 다른 이름으로 재등장해도 기록칩은 그대로 쌓임
- UI: 칩 모양 아이콘. 클릭 시 프로필 팝업 위에 기록 팝업 오버레이

### 기록 팝업 구조

**상단 — 수치 기록**

| 항목 | 설명 |
|------|------|
| 총 탐사 동행 횟수 | 이 얼굴과 함께한 탐사 수 |
| 총 처치 이형 수 | 누적 킬 카운트 |
| 총 사망 횟수 | 영구사망 당한 횟수 |
| 최고 도달 구역 | 역대 가장 깊이 들어간 구역 |
| 최고 코그 등급 | 역대 가장 높았던 코그 |
| 베테랑 여부 | 탐사 10회 이상 생존 시 마킹 |
| 첫 영입 Day | 처음 영입된 날 |

**하단 — 사망 연대기**

최신 사망이 위로 오는 역순 정렬. 건조한 텍스트.

```
1번째 사망
Day 14 · Cog 5 · 라운드 3 · 야생 AI에게 사망

2번째 사망
Day 3 · Cog 2 · 라운드 1 · 심해어에게 사망
```

### 게임적 효과

수치 전부 보존. 추후 구체화 단계에서 아래 항목 기준으로 효과 설계 예정.

- 탐사 동행 횟수 → 스탯 보정 or 영입 비용 감소
- 처치 수 → 공격 관련 보정
- 사망 횟수 → 영입 비용 감소 or 특수 반응
- 베테랑 여부 → 사용 스킬 관련 보정
- 최고 코그 → 스탯 상한 보정 참조값

### 설계 의도

기록칩의 핵심은 수치 보정이 아니라 플레이어가 그 얼굴에 서사를 투영하게 만드는 것.
"17번째 사망" 기록을 달고 스타팅 목록에 다시 나타나는 얼굴을 또 고를 때의 감정이 게임의 정체성.

## 4. 탐사 루프

### 전체 흐름

```
[탐사 난이도 슬롯] — 3개 중 1개 선택
    ↓
[파티 구성]
    ↓
[라운드 슬롯] ─────────────────────┐
    ↓                              │
[전투 / 이벤트 / 상점]             │
    ↓                              │
[일시 결산]                        │
    ↓                              │
[다음 라운드] ──────────────────────┘ (최대 5라운드)
    ↓
[최종 결산] → Day 증가
```

### 탐사 난이도 슬롯

- 슬롯머신으로 난이도 3개 중 1개 선택
- 난이도 = 코그 등급(1~10)으로 표기
- 선택한 난이도 = 해당 탐사의 코그 상한
- 코그 상한 초과 캐릭터는 파티 편성 불가
- 난이도 높을수록 적 스탯↑ 보상↑

### 라운드 슬롯 결과 목록

| 결과 | 설명 |
|------|------|
| 전투 | 이형과 전투 진행 |
| 낚시 이벤트 | 낚시꾼 활용 자원 수집 |
| 잔해 수집 이벤트 | 잠수부 활용 고철/아이템 수집 |
| 전선 연결 이벤트 | AI 활용 화폐 수집 |
| 심해 상점 | 심해화폐로 아이템 구매 |
| 표류체 발견 | 즉발형 도박 (자원 or 함정) |
| 추적 신호 | 선택형 도박 (고보상 or 기습전투) |
| 긴급 정비 | 잠수정 체력 소량 회복 |
| 철수 | 탐사 즉시 종료 후 결산 |

**철수 슬롯 강제 포함 조건**
- 파티 캐릭터 절반 이상 전투불능
- 잠수정 체력 50% 이하
- 위 조건 충족 시 다음 라운드 슬롯 3개 중 1개는 반드시 철수

### 스타팅 캐릭터

- 매판 시작 시 2~3명 중 1명 선택
- 영구사망 가능. 죽어도 다음 판 스타팅 목록 or 영입 목록에 재등장
- 기록칩 영구 누적 → 가장 기록이 많이 쌓이는 캐릭터가 됨

## 5. 전투 시스템

### 화면 구성

- 상단: 적 이형 일러스트
- 하단: 아군 캐릭터 초상화 배열
- 자동전투 기반
- 플레이어 개입: 게이지 충전된 캐릭터 초상화 클릭 → 사용 스킬 발동

### 전투 진행

- 전투 시작 시 이형 등장
- 아군 캐릭터 자동 공격
- 처치 발생 시 직업별 보상 처리
- 잠수정 체력 0 or 전원 전투불능 → 탐사 실패 → 전원 영구사망

### 이형 종류 (초안)

| 이형 | 행동 패턴 |
|------|-----------|
| 심해어 | 기본형. 일반 공격 |
| 야생 AI | 빈틈 공략형. 취약한 대상 우선 공격 |
| 침수자 | 웨이브형. 다수 등장, 주변 침수자 수에 따라 스탯 증가 |

### 캐릭터 사망 처리

| 상태 | 조건 | 처리 |
|------|------|------|
| 전투불능 | 체력 0 | 라운드 종료 시 회수 가능 |
| 회수 | 라운드 종료 + 잠수정 생존 | 이후 전투 불가, 경영 단계에서 자원으로 회복 |
| 영구 사망 | 탐사 실패 | 캐릭터 데이터 삭제 |

## 6. 이벤트 시스템

전투 화면 그대로 사용. 적 대신 오브젝트가 등장하는 방식.

### 잔해 수집

- 고철 더미, 침몰 컨테이너 등 오브젝트 등장
- 잠수부가 일정 시간 접근 시 채굴 완료
- 다른 직업 배치 시 채굴 속도 대폭 감소

### 전선 연결

- 끊어진 전선 오브젝트 등장
- AI가 접근하면 복구 진행
- 완료 시 화폐 수집

### 낚시

- 미니게임 방식
- 타이밍 기반 클릭
- 낚시꾼이 파티에 있을수록 성공 범위 확장

### 공통 규칙

- 직업 일치 캐릭터: 수집 효율 100%
- 직업 불일치 캐릭터: 수집 효율 대폭 감소
- 이벤트 결과도 결산에 합산

## 7. 결산 시스템

### 일시 결산 (라운드 종료마다)

- 획득 자원 (고철, 화폐, 아이템, 심해화폐)
- 캐릭터 현재 체력 및 상태
- 게이지 현황 (유지됨)

### 최종 결산 (탐사 종료)

- 탐사 전체 획득 자원 합산
- 캐릭터 최종 상태 반영
  - 회수 캐릭터: 경영 단계에서 자원 소모 후 회복
  - 영구 사망: 데이터 삭제
- 기록칩 수치 업데이트
- 게이지 초기화
- Day 카운트 증가 → 경영 단계로 전환
