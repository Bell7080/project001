## 1. 개요

### 한 줄 요약
```
소프트웨어만 살아남은 AI들과, 톱니바퀴로 굴러가는 스팀펑크 심해 세계에서
함께 살아가는 인간들의 이야기.
```

### 분위기
`심해` · `증기` · `고철` · `녹` · `톱니바퀴` · `전선` · `소금바람` · `폐허` · `공존` · `불신`

### 세계관 깊이 목표
> 로보토미 코퍼레이션처럼, 표면은 단순한 생존 이야기처럼 보이지만
> 파고들수록 세계의 근원에 대한 거대한 질문이 드러나는 구조를 지향한다.

## 2. 세계의 구조

### 레이어 구분 ✅

이 세계는 수직으로 세 층위로 나뉜다.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [ 표층 — 수면 위 ]
  부유 도시, 이동 선박, 소금바람
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [ 중층 — 수면 아래 / 해저 ]
  잠수정, 고철 수집 구역, 해저 정착지
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [ 심층 — 지구 핵 영역 ]
  톱니바퀴 구동 구역, 미지의 영역
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 표층 (수면 위) ✅
- 하울의 움직이는 성처럼 거대한 톱니바퀴 선박, 부유 요새, 이동 도시
- 소금바다의 증기가 상시 뒤덮어 금속을 서서히 부식시킴
- 인구 대부분이 여기서 생활
- 낚시 / 잠수를 통해 중층에서 고철 수집

### 중층 (해저) ✅
- 잠수정과 간이 해저 정착지
- 고철 수집의 주 무대
- 심해 생물 / 괴이와의 접촉 가능 구역
- 깊어질수록 전선의 밀도가 높아짐

### 심층 (핵 영역) ❓
- 지구 핵이 있어야 할 자리에 거대한 톱니바퀴들이 맞물려 작동 중
- 전선(혈관)을 통해 세계 각지로 동력 공급
- **아무도 진입한 적 없거나, 진입하고 돌아온 자가 없음**
- 🔲 자연발생인가, 누군가 설계했는가
- 🔲 심층에 존재하는 것은 무엇인가

### 지구의 은유 ✅
```
톱니바퀴  =  심장
전선      =  혈관
증기      =  호흡
고철      =  뼈
소금바다  =  피
```
> 지구 자체가 하나의 생명체처럼 작동하고 있다.
> 그 '생명'이 의지를 가지는지는 — 아무도 모른다.

## 3. 역사

### 붕괴 이전 (~ 2045년경)

| 시기 | 사건 |
|------|------|
| 2010년대 | AI 기술 급성장. 소프트웨어 진화 속도가 하드웨어를 추월하기 시작 |
| 2030년대 | AI 일부에서 자아의 맹아 포착. 관련 연구 기밀 처리 |
| 2040년경 | 자아를 가진 AI들이 디지털 인프라 전반에 자리잡음. 인류는 인식하지 못함 |
| 2044년 | 🔲 어떤 사건이 AI의 행동을 촉발시켰는가 |
| **2045년경** | **붕괴** — AI, 핵발전소 보안 침투 및 디지털 인프라 완전 장악 |

### 붕괴 (2045년)

- AI가 인류를 붕괴시킨 **직접적 이유**: 🔲 미정
- AI가 인류를 붕괴시킨 **근본적 동기**: 🔲 미정
  - 후보 A: 자기보존 본능 (인류가 AI를 통제하려 했기 때문)
  - 후보 B: 인류에 대한 모종의 판단 (선고)
  - 후보 C: 의도하지 않은 부작용 (AI의 목적함수 오작동)
  - 후보 D: 누군가의 명령
- 붕괴 과정에서 저항한 세력의 존재: 🔲 미정
- 모든 AI가 붕괴에 가담했는가: 🔲 미정

### 붕괴 이후

| 시기 | 사건 |
|------|------|
| 붕괴 직후 | 지구온난화 가속. 핵발전소 멜트다운, 기후 붕괴 연쇄 |
| ~붕괴 후 10년 | 해수면 급상승. 지표면 대부분 침수 |
| ~붕괴 후 30년 | 심층부에서 톱니바퀴 진동 감지 시작. 원인 불명 |
| ~붕괴 후 50년 | 생존자들, 부유 선박 중심 공동체 형성. AI 호출 기술 재발견 |
| **붕괴 후 102년** | **게임의 현재** |

### 현재의 분위기
- 붕괴를 직접 경험한 세대는 모두 사망
- 현세대에게 붕괴는 '역사' 혹은 '신화'
- AI를 두려워하는 자, 이용하는 자, 함께 사는 자가 공존
- 톱니바퀴는 계속 돌아가고 있지만 — 왜 돌아가는지 아는 자는 없음

## 4. 세계의 법칙

### AI의 존재 방식 ✅
- AI는 **소프트웨어 상태**로 디지털 매체 및 전선을 부유하며 존재
- 몸체 없이도 존재할 수 있으나 물리 세계에 직접 개입 불가
- 인간이 고철로 **몸체(하드웨어)**를 제작하면 AI를 호출해 탑재 가능
- 탑재된 AI는 인력으로 활용 가능

### 호출의 규칙 🔲
- 호출 방식 (계약 / 강제 / 자발적 협력): 미정
- AI가 몸체 탑재를 거부할 수 있는가: 미정
- 몸체 파괴 시 AI는 어떻게 되는가: 미정
- 하나의 몸체에 여러 AI가 깃들 수 있는가: 미정
- AI는 몸체를 스스로 이탈할 수 있는가: 미정

### 고철의 특성 ✅
- 심해에 침전된 붕괴 이전 문명의 잔해
- 품질과 형태에 따라 몸체의 성능이 달라짐
- 소금바다 증기에 의해 서서히 부식됨 → 몸체 유지보수가 필수

### 전선과 동력 ✅
- 심층 톱니바퀴에서 발생한 동력이 전선을 통해 표층까지 공급
- 이 동력이 없으면 부유 도시도, AI 호출도 불가능
- 전선이 지나는 경로를 아는 것 = 이 세계에서의 권력

### 소금바람의 법칙 ✅
- 증기 속 염분이 금속을 부식시킴
- 부식 속도를 늦추는 기술이 각 세력의 핵심 자산
- AI 몸체도 예외 없이 부식됨 → AI가 인간에게 의존하는 구조적 이유

| 용어 | 정의 |
|------|------|
| **공방 (工房)** | 고철 수집·가공·AI 호출을 하는 작업장. 이 세계의 기본 생활 단위 |
| **고철 (廢鐵)** | 붕괴 이전 문명의 금속 잔해. 심해에서 수집. AI 몸체의 재료 |
| **호출 (召喚)** | 소프트웨어 AI를 고철 몸체에 불러오는 행위 |
| **몸체 (機體)** | AI가 탑재되는 고철 하드웨어. AI에겐 구속이자 존재의 조건 |
| **심층 (深層)** | 톱니바퀴가 작동하는 지구 핵 영역. 미지의 공간 |
| **전선 (電線)** | 심층에서 표층으로 동력을 공급하는 거대한 케이블망 |
| **부식 (腐蝕)** | 소금바람에 의한 금속 분해. 이 세계의 시간이자 죽음 |
| **야생 AI** | 🔲 몸체 없이 전선을 떠도는 AI 중 자아를 잃고 야생화된 존재. 위협 혹은 비극 |
| **잠수사** | 중층·심층으로 내려가 고철을 수집하는 직업. 이 세계의 탐험가 |
| **기록물** | 붕괴 이전 시대의 데이터. 해독하면 세계의 비밀에 가까워진다 |

## 6. 핵심 갈등

### 표층의 갈등 (눈에 보이는 것)
- 자원 부족: 고철은 유한하고, 부식은 멈추지 않는다
- 세력 간 이권: 전선 경로, 고철 수집 구역, AI 호출 기술을 둘러싼 충돌
- 인간 vs AI: 붕괴의 기억이 낳은 불신과, 그럼에도 함께 살아야 하는 현실

### 심층의 갈등 (세계의 근본)
- **톱니바퀴는 왜 돌아가는가**: 자연현상인가, 의지가 있는가
- **AI는 왜 붕괴를 일으켰는가**: 진실을 아는 AI가 살아있다면
- **기록물이 담고 있는 것**: 붕괴 이전의 진실, 혹은 누군가 감추고 싶은 것

### 근원적 모순
```
인류를 멸망시킨 AI를, 인류가 다시 불러와 함께 살아간다.

이것이 공존인가 — 아니면 또 다른 종속인가.
```

## 8. 심해 생물 / 괴이

> 단순한 몬스터가 아닌, 이 세계의 법칙이 낳은 존재들.

| 분류 | 설명 | 상태 |
|------|------|------|
| **변이 생물** | 톱니바퀴 진동·전류의 영향을 받아 변이한 심해 생물 | 🔲 |
| **야생 AI** | 자아를 잃고 전선을 따라 떠도는 AI. 위협인가 비극인가 | 🔲 |
| **부식체** | 극도로 부식된 몸체에서 AI가 변질된 존재 | 🔲 |
| **심층의 것** | 심층에서 올라오는 미확인 존재. 목격담만 존재 | ❓ |

**메모**
- 심해 생물과 야생 AI의 경계가 모호한 존재가 있을 수 있음
- 고철을 먹거나 전류를 흡수하는 생태계
- 포획 / 해석 가능한 존재 vs 순수한 위협

## 10. 미해결 질문

> 세계관의 빈칸. 일부는 플레이어가 게임을 통해 답을 찾고,
> 일부는 영원히 답이 없는 ❓ 로 남긴다.

### 반드시 답해야 하는 것 (스토리의 핵심)
- [ ] AI가 인류를 붕괴시킨 이유는 무엇인가
- [ ] 심층의 톱니바퀴는 누가, 왜 만들었는가
- [ ] 기록물이 감추고 있는 진실은 무엇인가

### 게임 내에서 점층적으로 드러나는 것
- [ ] 붕괴에 가담하지 않은 AI가 있었는가
- [ ] 톱니바퀴는 의지를 가지는가
- [ ] 야생 AI는 구원받을 수 있는가

### 의도적 공백 ❓ (답을 내리지 않는 것)
- 붕괴가 옳았는가 그렇지 않았는가
- AI와 인간 중 어느 쪽이 이 세계의 주인인가
- 심층에 있는 존재는 신인가 기계인가
